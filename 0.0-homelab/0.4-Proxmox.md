## 0.4.1 Certificate

Since we're running our own STEP CA, the procedure to obtain a self-signed certificate is somewhat more nuanced.

### Steps

1. Step CA root certificate and bind9 TSIG key need to be imported. 
TSIG key need to be placed on `/var/tmp` and owned by `nobody:nogroup` to work properly.
To import root certificate either;
    - Installing step-cli and bootstrapping the certificate
    - ```
        mkdir -p /usr/local/share/ca-certificates/custom
        cp root_ca.crt /usr/local/share/ca-certificates/custom/
        update-ca-certificates

2. We need to add the acme account via CLI `pvenode acme account register <account_name> <email_address> --directory <CA_URL>`

3. The rest of the steps are quite straight forward and done on the GUI.
