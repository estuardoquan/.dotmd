## 0.3.1 Certificate

Since we're running our own STEP CA, the procedure to obtain a self-signed certificate is somewhat more nuanced.

### Steps

- Step 1: Step CA root certificate must be imported. This can be done through the web GUI, copy/paste certificate and add to store.

- Step 2: We've written a shell script that will execute whenever the certificate is about to expire.
            - The script was taken from acme.sh [dns_nsupdate.sh](https://github.com/acmesh-official/acme.sh/blob/master/dnsapi/dns_nsupdate.sh)
            - `tsig-keygen -a HMAC-SHA512 key-name > /path/to/key` need to be generated and added to bind9. Manually import onto TrueNAS.
            - Verify that `NSUPDATE_KEY`, `NSUPDATE_SERVER`, `NSUPDATE_ZONE` variables are correct.
            
- Step 3: The rest of the steps are quite straight forward and done on the GUI.
